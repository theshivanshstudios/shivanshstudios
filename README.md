# Shivansh Studios

Official website for **Shivansh Studios** — luxury wedding & pre-wedding photography in
Jaipur, Rajasthan. A fast, static site hosted on GitHub Pages at
[shivanshstudios.com](https://shivanshstudios.com).

- **Edit the site:** see [`EDITING-GUIDE.md`](EDITING-GUIDE.md) — written for non-developers.
- **Tech:** plain HTML/CSS/JS, no build step. Photos optimised via `tools/optimize_images.py`.
- **Publish:** push to `main` → GitHub Pages deploys automatically.
